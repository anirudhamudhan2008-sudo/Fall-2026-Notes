* How to Use

** Publishing Notes

To publish notes to HTML and PDF formats:

#+BEGIN_SRC bash
# Start Emacs server (if not running)
emacs --daemon &

# Publish HTML
make html

# Publish PDFs
make pdf

# Publish everything
make all
#+END_SRC
