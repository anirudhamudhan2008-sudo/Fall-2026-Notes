3. How to Use

3.1. Publishing Notes

To publish notes to HTML and PDF formats: 

# Start Emacs server (if not running)
emacs --daemon &

# Publish HTML
make html

# Publish PDFs
make pdf

# Publish everything
make all

3.2. Viewing Notes

  • Web version: Visit GitHub Pages
  • PDF version: See PDFs/ directory
  • Org mode: Open .org files directly in Emacs
